# Contexto del Proyecto: Autonomek Web & IA

## ¿Qué es Autonomek?
Agencia digital colombiana de automatización e inteligencia artificial. Tagline: **"Tecnología Implacable"**.

## Propuesta de Valor
Convertimos el flujo de WhatsApp, redes sociales y sitio web de un negocio en una máquina de adquisición de clientes. Creamos empleados de ventas inteligentes que responden, cotizan y agendan citas 24/7 de forma autónoma.

## Público Objetivo
- Clínicas, odontólogos, veterinarias, spas (agendamiento de citas)
- Inmobiliarias y constructoras (calificación de leads)
- Talleres automotrices y concesionarios
- Hoteles y academias
- Tiendas e-commerce (Shopify)
- Logística y distribuidores
- Empresas B2B que necesitan prospección comercial

## Servicios Ofertados
1. **Empleado de Ventas IA** (producto estrella) — WhatsApp con atención 24/7, cotiza, agenda, envía links de pago
2. **Prospector IA B2B** — Motor de generación de leads B2B automatizado
3. **Recepcionista IA** — Reserva y gestión de citas (consultorios, clínicas, spas)
4. **Servicio al Cliente IA** — Sistema omnicanal con base de conocimientos vectorizada (RAG)
5. **Automatización Comercial** — Conexión del stack comercial (formularios, Sheets, Gmail, CRM/ERP)
6. **Desarrollo Web Premium** — Sitios ultrarrápidos con SEO, chat IA, WhatsApp y CRM integrados

- **Backend:** Node.js, Supabase, PostgreSQL
- **IA:** OpenAI (GPT, Vision, Embeddings), RAG con vectores
- **Automatización:** n8n, Apify, Firecrawl
- **Infraestructura:** Vercel, Google Cloud, servidores cloud
- **Integraciones:** WhatsApp Cloud API, Chatwoot, Google Sheets, Gmail, HubSpot, Telegram, ElevenLabs
- **Analytics:** Google Analytics 4 (G-W5P7806J2F)
